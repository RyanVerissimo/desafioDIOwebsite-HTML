Estrutura Básica do HTML

O que significa HTML?
HTML é uma linguagem de marcação usada na criação de páginas web.
Ela define a estrutura e o conteúdo de uma página. Também faz toda a parte interativa com o usuário.
Como consiste uma página HTML?
  1. HTML

    2. Dentro dessa tag, irá conter outras duas tags essênciais para a criação do site(Head e Body)

  2. HEAD

    3. Sendo a cabeça do HTML, o head é usado para conter informações essênciais do site, podendo conter títulos e links para usualidades externas.

  3. BODY

    4. Conhecida como o corpo da página, é nele onde todo será feito todo o conteúdo visível para o usuário

Principais Tags HTML

  • a = Criação de um Link.
  • blockquote = Usado para citações, apresenta um espaçamento.
  • br = Quebra de linha.
  • h1 = Define o título principal da página.
  • h2 = Define o segundo título principal da página. Sua hierarquia chega até o "h6".
  • hr = Criação de uma linha horizontal na página.
  • i = Modifica o texto para itálico.
  • img = Utilizada para colocar imagens na página.
  • input = criação de um formulário interativo que pode ser receber dados em forma de texto ou números.
  • mark = Marca uma parte do texto da página.
  • p = Usado para fazer um parágrafo na página.
  • small = Diminui o texto comparado ao texto padrão.
  • sub = Palavra que ficará abaixo de outra palavra.
  • sup = Palavra que ficará acima de outra palavra.
  • u = Usado para sublinhar parte de um texto ou palavra.
  • ol = criação de uma lista ordenada
  • ul = criação de uma lista não ordenada

Atributos da Tag

Alguns exemplos:

  • id: para identificação única, podendo passar por chamadas no CSS e JS.
  • style: Sendo usado mais normalmente no css, no html, também, pode ser usada para mudar cores e tamanho de palavras o textos.
  • class: Usado para nomear uma parte do código, para ser chamado no CSS e JS para futuras interações.
  • type: Indica o tipo de conteúdo associado. Na tag input, o atributo "type" serve para definir o tipo de entrada, se vai ser do tipo numérico ou texto.
  • src: Usado para passar o endereço de uma imagem, assim ela aparecerá no site.

Listas Ordenadas e Não Ordenadas

Listas Ordenadas, mais conhecidas como "ol", ordenam elementos de acordo com números em forma crescente.
Exemplo:
  1. Arroz
  2. Banana
  3. Cuscuz
Listas Não Ordenadas, mais conhecidas como "ul", ordenam elementos de forma em que suas posições não importam.
Exemplo:
  • Fortnite
  • Valorant
  • Rocket League

Links

Usa-se a tag "a" para a criação de links, onde essa tag pode conter atributos que irão influenciar no caminho das páginas.
Exemplo:
A Tag "Target" pode receber dois parâmetros:
_blank: ao clicar no link, irá abrir outra aba com o caminho deste link.
_self: comportamento padrão, onde abrirá o link na mesma aba.
